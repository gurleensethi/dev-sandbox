# dev-sandbox

Quickly run predefined sandbox environments inside docker containers.

# Install

```bash
go install github.com/gurleensethi/dev-sandbox@latest
```

# Usage

```bash

dev-sandbox run [nodejs|golang|react]

dev-sandbox list

```
